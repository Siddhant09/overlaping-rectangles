The source code contains of two java files, a makefile, a readme file, and three input files inside InputFiles folder.

Command to compile the code:
make

Command to run the program:
make program

Command to delete the existing class files:
make clean

When the program executes, a choise will be give to the user to execute any one of three input files.
Upon selecting an option, the selected file will be executed and result will be displayed.
